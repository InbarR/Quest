# Keyboard Shortcuts

Master Quest with these essential shortcuts.

## Query Execution

| Shortcut | Action |
|----------|--------|
| **F5** | Run query |
| **Ctrl+Enter** | Run query |
| **Shift+F5** | Cancel query |

## Editor

| Shortcut | Action |
|----------|--------|
| **Ctrl+Space** | Show IntelliSense |
| **Ctrl+Shift+S** | Save as favorite |
| **Ctrl+/** | Toggle comment |

## Navigation

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+M** | Switch query mode |
| **Ctrl+Shift+A** | Open AI Assistant |
| **Ctrl+Alt+P** | Switch connection profile |
| **F1** | Show all commands |

## Results

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+C** | Copy results |
| **Ctrl+Shift+E** | Export results |

## Customization

You can customize any shortcut in VS Code:

1. Open **File > Preferences > Keyboard Shortcuts**
2. Search for "Quest"
3. Click the pencil icon to change a binding

Happy querying!
