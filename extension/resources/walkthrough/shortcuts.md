# Keyboard Shortcuts

Master Quest with these essential shortcuts.

## Query Execution

| Shortcut | Action |
|----------|--------|
| **F5** | Run query at cursor |
| **Shift+F5** | Cancel running query |
| **F1** | Show keyboard shortcuts dialog |

## Editor

| Shortcut | Action |
|----------|--------|
| **Ctrl+Space** | Show IntelliSense completions |
| **Ctrl+Shift+S** | Save query to favorites |
| **Ctrl+/** | Toggle line comment |
| **Ctrl+Shift+/** | Toggle block comment |

## Navigation

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+M** | Switch query mode (Kusto/ADO/Outlook) |
| **Ctrl+Shift+A** | Open AI Chat panel |
| **Ctrl+Shift+P** | VS Code command palette |

## Results Panel

| Shortcut | Action |
|----------|--------|
| **Ctrl+C** | Copy selected cells/rows |
| **Ctrl+A** | Select all rows |
| **Home** | Scroll to top |
| **End** | Scroll to bottom |
| **Escape** | Clear selection |

## Results Table Mouse Actions

| Action | Effect |
|--------|--------|
| **Click row** | Select single row |
| **Ctrl+Click** | Add/remove row from selection |
| **Shift+Click** | Select range of rows |
| **Double-click cell** | Open link/item or copy value |
| **Right-click** | Context menu with options |
| **Click column header** | Sort by column |

## Filter Box

| Key | Action |
|-----|--------|
| **Enter** | Apply filter |
| **Escape** | Clear filter |
| **Ctrl+F** | Focus filter box |

## Customization

Customize shortcuts in VS Code:

1. **File → Preferences → Keyboard Shortcuts**
2. Search for "Quest" or "queryStudio"
3. Click the pencil icon to rebind

Or edit `keybindings.json` directly for advanced customization.
