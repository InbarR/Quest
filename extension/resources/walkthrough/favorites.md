### History Features

- Auto-saves every executed query
- Shows execution time and row count
- Grouped by date (Today, Yesterday, This Week, etc.)
- Click to reload and re-run

### History Item Actions

| Option | Description |
|--------|-------------|
| **Load Query** | Open in editor |
| **Run Query** | Execute again |
| **Save to Favorites** | Promote to favorite |
| **Copy Query** | Copy to clipboard |
| **Delete** | Remove from history |

## Saved Results Panel

View past query results:

- Results are saved automatically
- Click to view the result data
- Compare results across runs
- Export historical data
