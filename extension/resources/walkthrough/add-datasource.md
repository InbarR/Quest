# Add a Data Source

Configure your data sources to start querying.

## Kusto Mode

Add your Azure Data Explorer cluster:

1. Click **Add Data Source** in the sidebar
2. Enter your cluster URL:
   - `https://help.kusto.windows.net` (sample data)
   - `https://yourcluster.kusto.windows.net`
3. Select a database from the list

**Tip:** The `help` cluster has sample data for learning KQL!

## ADO Mode

Add your Azure DevOps organization:

1. Click **Add Data Source** in the sidebar
2. Enter your organization URL:
   - `https://dev.azure.com/yourorg`
3. Select a project from the list

## Outlook Mode

No configuration needed! Outlook mode automatically connects to your local Outlook installation.

Just switch to Outlook mode and start querying your emails, calendar, and contacts.
